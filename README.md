# 💰 Personal Finance Tracker

A personal finance tracking app built with Python and Streamlit.

## Features
- 🔐 User authentication (sign up / sign in)
- 💱 Multi-currency support (USD, EUR, BRL)
- 📊 Income, fixed and variable expense tracking
- 📈 Interactive dashboard with charts
- 🌐 Available in English, Português and Español

## Live App
👉 [Access the app](https://personal-finance-tracker-hbe6ghpujtecfvqy34s5lh.streamlit.app/)

## Tech Stack
- Python
- Streamlit
- Supabase (auth + database)
